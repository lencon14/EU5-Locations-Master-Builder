\# EU5 Locations Master Builder



\## Purpose

Extract factual location data from Europa Universalis V assets.

This script produces a CSV used downstream by Urbanization Analyzer.



\## Principles

\- CSV schema must not change

\- This tool extracts \*facts\*, not balance or evaluation

\- Lake adjacency is determined from map images (primary source)



\## Requirements

\- Python 3.x

\- Europa Universalis V installed locally



\## Usage

```powershell

python src/eu5\_locations\_master\_builder.py
